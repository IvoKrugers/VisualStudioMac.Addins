Add-in Repository

Visual Studio Mac 2022 Addins by Ivo Krugers

This is a list of add-ins available in this repository.

Add-inVersionDescription
VSM: Generate .mpack17.5.1This extension adds a item to the tools menu to generate a .mpack file from a release build. By Ivo Krugers
SimplePad demo17.5Demo how to create a pad. by Ivo Krugers
VSM: Get Git Repo URL of selected lines17.5.1This extension copies the URL of the selected lines in the connected Git Repo to the clipboard. GITHUB and Azure DevOps are supported. by Ivo Krugers
VSM: SolutionTree Filter17.5.1This extension adds a Pad which allows you to filter the solution tree by multiple search terms. By Ivo Krugers
VSM: One Click To Open File17.5.1This extension opens a file with only one click on the solutiontree. by Ivo Krugers
